結論 — VERDICT

D3 の方向性に同意するが、そのままでは採用しない。実測で D3 に「カレント塗りの上の blocked グリフが CR 3.08:1」という設計思想と矛盾する穴が見つかった。最重要信号の blocked が、ユーザーが最も注視する場所（カレント session）でだけ最弱になる。

推奨は D3改：カレントのグリフを塗りの外（バー地の上）に出す。これで全状態グリフが例外なくバー地上に載り、D3 が掲げた「全所で単一パレット」がむしろ完全達成される。tmux フォーマット上は単なる順序の入れ替えで済む。

ゼロベースで導いた構造原則 暗いバー上で「状態色の全所一貫」と「アクティブの強調」を両立する構造は一つしかない — 信号グリフは常に暗い地の上に置き、塗り（明るい面）には信号を載せない。明るい pill と単一状態色は輝度的に両立不可能（現行の CR 1.20 はその帰結）。D3 系はこの原則に沿うが、塗り上グリフの1点で自らの原則を破っている。

§5案とも見落としている論点

1breadcrumb との輝度衝突

バー地を #232332 → #1a1926 に落とすと breadcrumb 地 #181824 とほぼ同輝度になり、現在成立している「breadcrumb は一段暗い面」という区別が消える。D3 / D3改 / D1 採用時は breadcrumb 地の再調整（例 #121218）が必須の随伴作業。

2色覚多様性は形状が救っている

blocked(赤) / working(緑) は deuteranopia で最悪の組だが、▲●✓○ の形状冗長性が既に耐性を担保している。将来グリフ形状を変える変更でこの冗長性を壊さないよう、設計制約として明文化すべき。

3痛点②の根本原因は ANSI 名前色

グリフが淡いのは mod.rs が "green" 等を出力し、ターミナルテーマのパステル ANSI パレットに写像されるため。どの案でも hex 直書きか設定化が必須の共通作業。ただし現行の明 pill 上では暗色 hex が要り二重パレット化するので単体適用は不可。

4truecolor 前提の明文化

hex 指定は terminal-overrides で Tc/RGB が有効でないと 256 色近似に落ち、微妙な暗色差（案B の chip 群など）は全滅する。採用案の前提条件としてドキュメント化が必要。

§実装観点の懸念 — src/statusline/mod.rs

  1. グリフ色は設定化すべき。

    badge_fragment / category_badge_fragment（mod.rs:216-221, 321-326）に色がハードコードされているが、pill 色は既に config 化されており非対称。hex を再ハードコードするより badge.colors.{blocked,working,done,idle}（既定値 = D3 の hex）として badge.glyphs と対に置くのが一貫する。

  2. restore 機構が D3改と干渉する。

    badge_fragment はグリフ後に #[fg={style.colors.fg}] で復元する（mod.rs:329-330）が、グリフをセグメント外＝バー地に出す D3改では「グリフはバーの bg/fg → 矩形塗り」という並びになり、グリフとラベルが同一スタイル内という前提が崩れる。render_session_segment の {badge} トークン処理を「セグメント外プレフィックス」に昇格させる改修が要る。矩形採用でキャップ glyph の描画処理自体は不要になるため、改修の正味量はむしろ減る。

  3. underline 属性が未実装。

    tmux_style_segment は bold のみ対応に見える。D1/D3 系のカレント表現に使うなら、tmux では #[underscore]（underline ではない）の属性追加が config・レンダラ双方に必要。

  4. ラベル展開による幅ジッタ。

    アクティブ category のラベル展開は status-left の幅を動的に変え、category 切替のたびに window list の開始位置が揺れる。status-left-length の余裕確保と展開幅の上限（truncate）を決めておかないと、多 session 時に window list が押し出される。

  5. D1 の隠れコスト。

    D1 は「色だけの案」に見えるが、underline 対応・ラベル展開・塗り描画の除去分岐が必要で変更量は D3 とほぼ同等。変更量最小を優先するなら A だが、主訴解決の確度が低い。
